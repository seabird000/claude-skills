# 排版示例（输入 → 输出对照）

## 示例 1：无结构长文 → 加标题加粗

### 原文输入

```
人工智能的发展经历了几个重要阶段。第一阶段是符号主义时期，研究者试图用逻辑规则模拟人类思维。第二阶段是连接主义兴起，神经网络开始受到关注。第三阶段是深度学习爆发，2012年AlexNet在ImageNet竞赛中取得突破性成绩，标志着深度学习时代的到来。目前我们处于大语言模型时代，GPT、Claude等模型展现出惊人的语言理解和生成能力，正在重塑各个行业的工作方式。
```

### 排版输出

```markdown
# 人工智能的发展阶段

人工智能的发展经历了几个重要阶段。

**第一阶段**是符号主义时期，研究者试图用逻辑规则模拟人类思维。**第二阶段**是连接主义兴起，神经网络开始受到关注。**第三阶段**是深度学习爆发，**2012年** AlexNet 在 ImageNet 竞赛中取得突破性成绩，标志着深度学习时代的到来。

目前我们处于**大语言模型时代**，GPT、Claude 等模型展现出惊人的语言理解和生成能力，正在重塑各个行业的工作方式。
```

**改动说明：**
- 新增一级标题（原文无标题）
- 长段落按语义拆分为两段
- 加粗3处：第一/二/三阶段、2012年、大语言模型时代
- 未删减任何文字

## 示例 2：有 frontmatter → 清理元数据

### 原文输入

```
---
author: 张三
source: https://example.com
saved: 2026-01-01
tags: [AI, 深度学习]
id: abc123
---

# 深度学习简介

深度学习是机器学习的一个子领域……
```

### 排版输出

```markdown
# 深度学习简介

深度学习是机器学习的一个子领域……
```

**改动说明：**
- 删除了 frontmatter 块（`---` 包裹的全部元数据）
- 正文标题和内容完整保留

## 示例 4：叙述性并列 → 保持原样（不转列表）

### 原文输入

```
他既是一个优秀的工程师、也是一个热爱音乐的人、同时还在社区做志愿者。
```

### 排版输出

```markdown
他既是一个优秀的工程师、也是一个热爱音乐的人、同时还在社区做志愿者。
```

**说明：** 这是叙述性并列，不是功能/步骤枚举，保持原句不转列表。
